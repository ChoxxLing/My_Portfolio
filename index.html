My Project

→

The Student Recording System is a Java-based project developed for school management purposes. Its primary objective is to streamline the monitoring process of students within educational institutions. The system offers a user-friendly interface and supports three distinct roles: Admin, Teacher, and Student. The Admin role holds overarching control, managing user accounts and system security.

Additionally, Admins facilitate account creation, providing initial login credentials while users complete their account details. This ensures a secure and efficient user management process. Within the system, teachers can input students' overall averages, enabling easy access to grades for students. The system also incorporates birthday notifications and etc.
